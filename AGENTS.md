# TraceLoop

## Agent skills

### Issue tracker

Issues and specs (PRDs) live as local markdown files under `.scratch/<feature-slug>/` — no external tracker, no GitHub/GitLab round-trips. See `docs/agents/issue-tracker.md`.

### Domain docs

Single-context — one `CONTEXT.md` + `docs/adr/` at the repo root, created lazily by the domain skills. See `docs/agents/domain.md`.
